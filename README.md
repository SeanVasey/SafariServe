# SafariServe

An easy-to-use shortcut generator for non-Safari browsers on iOS.

## Overview

SafariServe helps iOS users create home screen shortcuts that open directly in their preferred browser (Chrome, Firefox, Brave, etc.) instead of defaulting to Safari.
